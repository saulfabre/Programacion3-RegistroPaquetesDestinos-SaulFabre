## Descripción
Aplicacion de escritorio desarrollada en JavaFX para el registro y consulta de paquetes y destinos, la aplicación permite almacenar los destinos y paquetes en archivos de texto, cargar los datos para su consulta y navegar entre distintas pantallas mediante una interfaz gráfica.


## Tecnologías utilizadas
- JavaFX
- FXML
- CSS
- JAVA
- JDK 25
- Visual Studio Code
